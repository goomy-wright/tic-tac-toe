# Tic Tac Toe Mini Console Game in C++
## Include Library
```cpp
#include "path/to/include/Game.h"
```

## Start the Game
```cpp
int main(int argc, const char *argv[])
{
    TicTacToe game;
    game.start();
}
```

## How to Play
* Input (1-9) - Field Number
